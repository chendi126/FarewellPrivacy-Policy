更新日期：2026年4月29日  
生效日期：2026年4月29日

本隐私政策适用于“经历·Farewell”应用及相关服务。  
本隐私政策的声明主体为：**陕西望鹊星河软件技术有限责任公司**。

## 一、我们收集和使用的信息

### 1. 账号注册、登录与安全

当您注册、登录或使用账号服务时，我们可能处理以下信息：

- 邮箱地址
- 登录密码（仅以加密/散列方式处理，不以明文长期保存）
- 邮箱验证码、密码重置验证码
- 昵称、头像等您主动填写的资料
- 登录时间、会话标识
- 登录时产生的网络信息，例如 IP 地址、User-Agent

处理目的：

- 完成账号注册、登录和身份验证
- 维持登录状态
- 找回密码
- 保障账号安全、防止异常登录

### 2. 地图、足迹记录与协作功能

当您使用地图记录、地点解析、协作地图等功能时，我们可能处理以下信息：

- 您授权的位置权限对应的地理位置信息
- 您创建的记录内容，例如标题、描述、地点名称、经纬度、地址信息
- 您创建或加入的协作地图信息，例如地图名称、简介、邀请码、成员关系
- 您在协作地图中上传的记录内容、附件路径、标记样式、签名等数据

处理目的：

- 地图展示与当前位置定位
- 地点搜索与逆地理解析
- 足迹记录展示
- 协作地图的创建、加入、展示与管理

### 3. 媒体采集与播放功能

当您使用拍照、相册选择、录音、媒体播放、保存图片到相册等功能时，我们可能处理以下信息：

- 您主动拍摄或选择的图片、视频、音频文件
- 相关文件路径、格式、大小、上传结果
- 您的签名图片

处理目的：

- 记录附件上传与展示
- 媒体播放
- 保存图片到系统相册
- 内容分享与导出

### 4. 支付与权益解锁

当您使用应用内支付或权益解锁功能时，我们可能处理以下信息：

- 当前登录的应用账号信息
- 订单号、商品 ID、支付状态、支付平台返回的订单信息
- 支付时间、支付平台标识、关联权益信息

目前本应用已接入**支付宝 App 支付客户端 SDK**用于商品解锁。我们处理上述信息，是为了实现：

- 商品购买
- 订单创建
- 支付结果确认
- 权益发放
- 订单状态追踪与安全审计

### 5. 本地存储与会话信息

为实现登录状态维持、偏好设置与页面展示，我们可能在本地安全存储或本地偏好存储中保存必要信息，例如：

- Access Token / Refresh Token
- 本地展示配置
- 本地缓存数据

### 6. 日志与故障排查

为了维护服务稳定、进行安全分析和异常排查，我们可能处理：

- 请求时间
- 接口访问结果
- 基础设备/应用环境信息
- 错误日志、崩溃信息、网络异常信息

## 二、设备权限使用说明

为了实现相关功能，本应用可能申请以下系统权限：

- 位置权限：用于地图展示、当前位置定位、地点解析
- 相机权限：用于拍照并创建记录
- 麦克风权限：用于录音
- 相册/媒体读取权限：用于选择图片、视频、音频文件
- 网络权限：用于登录、同步、上传、地图服务、支付结果确认

您可以在系统设置中关闭相关权限。关闭后，对应功能可能无法正常使用，但不会影响其他与该权限无关的功能。

## 三、信息共享、转让与公开披露

### 1. 信息共享

我们不会向无关第三方出售您的个人信息。  
在以下必要场景下，我们可能共享相关信息：

- 为实现地图服务、支付、媒体上传、系统能力调用所必需的第三方 SDK 或服务
- 根据法律法规、监管要求或司法机关要求进行披露
- 在您主动发起协作、分享、跳转系统能力时，为完成该功能所必需的范围内共享

### 2. 信息转让

除非获得您的单独同意，或符合法律法规规定，我们不会将您的个人信息转让给其他公司、组织或个人。

### 3. 公开披露

除法律法规另有规定或获得您单独同意外，我们不会公开披露您的个人信息。

## 四、我们如何存储和保护您的个人信息

1. 我们会在实现本政策所述目的所必需的最短期限内保存您的个人信息，法律法规另有规定的除外。  
2. 为了保护您的个人信息安全，我们采取了与业务规模相适应的技术和管理措施，包括但不限于：
   - **身份鉴别**：对登录、敏感操作、账号注销等行为进行账号身份校验
   - **数据加密**：对密码进行加密/散列处理；对登录令牌等敏感数据进行安全存储
   - **访问控制**：对后台管理接口、订单数据、用户资料等设置权限控制和最小必要访问原则
   - **恶意代码防范**：对应用依赖、服务端运行环境和发布包进行持续检查，减少恶意代码、异常依赖和高风险组件带来的安全风险
   - **安全审计**：保留必要的日志、订单状态变更记录和安全审计记录，用于异常排查和追踪
   - **订单与权益校验**：对支付结果、订单状态、权益发放进行服务端校验，防止伪造或篡改
3. 尽管我们已采取合理措施，但互联网环境并非绝对安全，请您妥善保管自己的账号和密码。

## 五、您的权利以及实现路径

在法律法规规定范围内，您享有对个人信息的查询、更正、删除、撤回授权和账号注销等权利。您可以通过以下路径行使相关权利：

### 1. 查询个人信息

您可以通过以下路径查询与您账号有关的个人信息：

- **查询账号基本信息路径**：`我的 -> 设置`
- **查询主页资料路径**：`我的 -> 我的主页`
- **查询协作地图与记录路径**：进入对应地图或记录详情页查看

### 2. 更正个人信息

您可以通过以下路径更正或补充个人信息：

- **更正主页资料路径**：`我的 -> 我的主页 -> 编辑主页`
- 可修改的内容包括：头像、背景、名称、简介等您主动填写的展示资料

如您需要更正应用内暂不支持直接修改的信息，可通过开发者平台公示的联系方式与我们联系。

### 3. 删除个人信息

您可以通过以下路径删除部分个人信息或内容：

- **删除记录路径**：进入记录详情页后选择“删除”
- **删除协作地图路径**：进入地图管理或地图详情页后选择“删除地图”

请您理解，删除后部分内容可能无法恢复；依法必须保留的交易记录、安全审计记录等信息除外。

### 4. 撤回授权

您可以通过系统设置关闭位置、相机、麦克风、相册等权限授权。关闭后，对应功能可能无法继续使用。

### 5. 注销账号

您可以通过以下路径发起账号注销：

- **注销账号路径**：`我的 -> 设置 -> 注销账号`

在发起注销时，应用会要求您输入当前密码进行确认。  
账号注销后，您将无法继续使用该账号登录服务；与该账号绑定的权益、协作关系、历史记录等可能无法恢复。依法必须保留的信息、交易结算信息及安全审计记录将按法律要求处理。

### 6. 响应时限

如您通过应用内路径或开发者平台公示联系方式提出个人信息相关请求，我们将在核验您的身份后尽快处理，并原则上在 **15个工作日内** 完成响应或给予说明。
